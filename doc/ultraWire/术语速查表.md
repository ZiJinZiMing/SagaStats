## 走线风格对照表

| 风格 | 实现位置 | 算法 | 复杂度 | 规则 |
|------|---------|------|--------|------|
| **Default (Bezier)** | 透传 UE 原生 `FConnectionDrawingPolicy` | 贝塞尔曲线 | O(1) | 保留 UE 默认曲线，但上层可叠加 Glow / Bubble 等效果 |
| **Manhattan (90°)** | `FUltraWireGeometry::ComputeManhattanPath` | 三段式直角 | O(1) | 水平出 → 垂直 → 水平进；反向连接用 U 形绕回 |
| **Subway (45°)** | `FUltraWireGeometry::ComputeSubwayPath` | 三段式 45° | O(1) | 水平出 → 45° 对角 → 水平进 |
| **Freeform** | `FUltraWireGeometry::ComputeFreeformPath` | 用户角度 | O(1) | 水平出 → 5°-85° 对角 → 水平进 |
| **Smart Routing** | `FUltraWireRouteEngine::ComputeSmartRoute` | A\* 寻路 | O(W² log W) with cache | 4 方向（Manhattan 启发）或 8 方向（Octile 启发）网格搜索，避开节点障碍 |

**转角样式**（所有非 Default 风格都可应用）：

| 样式 | 实现 | 描述 |
|------|------|------|
| **Sharp** | `ApplyCornerRounding` 跳过 | 保留 90° / 45° 硬角 |
| **Rounded** | 8 段圆弧逼近 | 半径可配 0-50 px，自动夹紧防重叠 |
| **Chamfered** | 45° 对角线替代 | 在折点处切角 |

## 效果系统对照表

| 效果 | 类 | 层级 | 每帧开销 | 缓存策略 |
|------|----|------|---------|---------|
| **Wire Lines（核心线段）** | `FUltraWireDrawingPolicy::DrawWireLines` | 主 LayerID | O(1) per wire | 无缓存 |
| **Glow（发光）** | `FUltraWireGlowRenderer::DrawGlowPass` | 背景 BackLayerID | 每线 4 条额外线 | 无缓存 |
| **Bubbles（流动气泡）** | `FUltraWireBubbleSystem::DrawBubbles` | 前景 FrontLayerID | 参数化采样 | 无缓存 |
| **Labels（自动标签）** | `FUltraWireLabelRenderer::DrawLabel` | 前景 FrontLayerID | 只在 Get/Set 变量线触发 | 无缓存 |
| **Crossings（交叉符号）** | `FUltraWireCrossingDetector::DetectCrossings` | 顶层 TopLayerID | O(W²·S²) + AABB 预裁剪 | 无缓存，帧尾聚合 |
| **Smart Routing（智能路由）** | `FUltraWireRouteEngine::ComputeSmartRoute` | — | A\* 网格搜索 | **路由缓存** + 50ms 超时 + 512×512 网格上限 + 按 AABB 增量失效 |
| **Theming（节点主题）** | `FUltraWireThemeEngine::ApplyTheme` | Slate 全局样式 | 仅配置变更时 | `OriginalBrushes` 快照缓存 |
| **Minimap（小地图）** | `SUltraWireMinimap` | 画布 overlay | 10 Hz 拓扑刷新 | 100ms 拓扑快照缓存 |
| **Heatmap（执行热度）** | `FUltraWireHeatmapBridge` | 注入 `Params.WireColor` | 4 Hz 轮询 | **零开销设计**：Profiler 未激活时完全不运行 |

## 支持的 12 种 Graph Type

Blueprint / Material / Niagara / Animation Blueprint / Behavior Tree / Control Rig / PCG / Sound Cue / MetaSound / Environment Query / Gameplay Ability / Other（fallback）

每种都可独立配置 `FUltraWireProfile`（通过 `GraphProfileOverrides: TMap<EUltraWireGraphType, FUltraWireProfile>`）。

## 内置预设

| 预设 | 特征 |
|------|------|
| **Default** | Manhattan + Rounded r=8，低饱和，适合日常 |
| **Circuit Board** | Sharp Manhattan + 方形 Pin + Gap 交叉 + Ribbon，PCB 美学 |
| **Neon Cyberpunk** | Default Bezier + 高强度脉冲发光 + 流动气泡 + 菱形 Pin，视频友好 |
| **Clean Professional** | Subway + 无动画 + 全不透明，文档截图友好 |
| **Retro Terminal** | Freeform + 慢速气泡 + 软发光，琥珀绿单色 |
